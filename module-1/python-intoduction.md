# Python Introduction

## What is Python?

Python is a high-level, general-purpose programming language designed to be simple and readable. Created by **Guido van Rossum** in 1989 and first released in **1991**, Python was built around one idea — code should read almost like plain English.

Today it is the **most popular programming language in the world**.

---

## Why Python?

Compare printing "Hello World" across languages:

**Java:**
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

**Python:**
```python
print("Hello World")
```

Less code. More clarity. Same result. This is Python's superpower.

## Where is Python Used?

| Field | What Python Does |
|---|---|
| **Web Development** | Powers backends of Instagram, Spotify, Reddit |
| **Data Science** | Cleans, analyses, and visualises large datasets |
| **Machine Learning** | Trains AI models — ChatGPT, image recognition, etc. |
| **Automation** | Replaces repetitive manual tasks with scripts |
| **Cybersecurity** | Builds tools for testing and protecting systems |
| **Research & Science** | Used by NASA, biologists, physicists worldwide |

---

## Python in the AI Era

A huge share of **AI research, model training, and ML tooling** is done in Python. Ecosystems like **TensorFlow**, **PyTorch**, and **scikit-learn** are central to how models are built and experimented with—often alongside other languages in production systems.

Learning Python helps you to:
- Read and sanity-check AI-generated code
- Call AI APIs from your own scripts
- Prototype features that use models or external AI services
- Collaborate in data and ML-heavy teams
